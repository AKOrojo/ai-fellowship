# AI Fellowships 2026

A community-maintained, always-current list of **AI fellowships, residencies, and funded programs** — industry residencies, research & safety cohorts, academic/PhD fellowships, and policy/grant programs.

> ⚠️ Deadlines change. Always confirm details on the official link before applying. Each entry shows when it was `last_verified`.

## How to contribute

- **Have a fellowship to add?** Open the [New fellowship issue form](../../issues/new?template=new-fellowship.yml) — no coding needed — or edit [`data/fellowships.yaml`](data/fellowships.yaml) and open a Pull Request.
- All submissions are validated automatically and **reviewed by a maintainer before merging**. Nothing is added automatically.
- See [CONTRIBUTING.md](CONTRIBUTING.md) for the field reference and [SECURITY.md](SECURITY.md) to report issues.

🔎 **Browse & filter the full list on the website:** see the repo's GitHub Pages site.
